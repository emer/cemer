2799
 
2008-04-19
 
oreilly
 
major optimization of spike mode in leabra: before it used exponential decay of
activation to simulate synaptic currents, requiring redundant sends of activations
(very expensive), now it just sends the activation once when it spikes and then
stores results in a moving time window buffer from which synaptic current is computed.
also now has a full alpha function for synatic current, including separate rise and 
decay time constants.  system is now fully integrate-and-fire compliant and 
we should get major speedup from using spikes.. if it will work with learning..
